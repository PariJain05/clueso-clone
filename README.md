## Overview
This project is a prototype clone of **Clueso.io**, developed as part of a software development internship assignment.  
It demonstrates the workflow of uploading a screen recording and generating structured step-by-step documentation.

## Features
- Upload a screen recording via a web interface
- Backend processing using Flask
- Display of step-by-step documentation (simulated for this prototype)
- Simple and clean UI

## Tech Stack
- Python
- Flask
- HTML / CSS

## How It Works
1. User uploads a screen recording.
2. The backend saves the file.
3. The system generates structured steps (simulated in this prototype).
4. The generated documentation is displayed in the browser.
